# Archived: MCP (Rube/Composio) Integration

**Archived:** MCP has been removed from Gerty. OpenClaw handles all app integrations (calendar, Gmail, Drive, Tasks, files, browser, etc.).

This directory preserves the MCP client code for reference. The code in `client.py` is a snapshot and will not run without the removed config vars (`COMPOSIO_API_KEY`, `RUBE_MCP_URL`) and the `mcp` Python package.

See [docs/OPENCLAW_INTEGRATION.md](../../OPENCLAW_INTEGRATION.md) for the current app integration path.
